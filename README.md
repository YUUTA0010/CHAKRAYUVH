# 🎮 Chanakya Neethi

## 📌 Overview
Chanakya Neethi is a 2D strategy-action web game inspired by the life of Chanakya (Kautilya). The game focuses on intelligent warfare, where players use strategy, planning, and decision-making to help Chandragupta Maurya defeat the powerful king Dhana Nanda.

---

## 🧠 Game Concept
The game combines:
- Strategy (planning, spies, alliances)
- Action (battle gameplay)
- Historical storytelling

Instead of brute force, the game emphasizes winning through intelligence and tactics.

---

## 🎭 Gameplay

### 🧠 Strategy Mode (Chanakya)
- Plan attacks
- Send spies
- Manage resources
- Build alliances

### ⚔️ Battle Mode (Chandragupta)
- Control the player in combat
- Fight enemies
- Navigate maze-style battlefield (Chakravyuh)
- Lead troops
